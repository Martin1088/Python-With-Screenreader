35. Flask Framework
Flask ist ein vom österreichischen Programmierer Armin Ronacher in Python geschriebenes Webframework. Der Fokus von Flask liegt auf Erweiterbarkeit und guter Dokumentation. (Quelle: www.wikipedia.de)
from flask import Flask
app = Flask('______name__')
@app.route('/') def index():
return ('<h1>Hallo Flask! </h1>')
@app.route("/hello/<name>") def hello(name):
return ('<h1>Hallo ' + name + '</h1>')
app.run(debug=True)
